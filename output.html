HTML:text/html, full-stop.

XHTML:application/xhtml+xml, or only if following HTML compatbility guidelines,text/html. See the W3Media Types Note.

XML:text/xml,application/xml(RFC 2376).

There are also many other media types based around XML, for exampleapplication/rss+xmlorimage/svg+xml. It's a safe bet that any unrecognised but registered ending in+xmlis XML-based. See theIANA listfor registered media types ending in+xml.

(For unregisteredx-types, all bets are off, but you'd hope+xmlwould be respected.)
